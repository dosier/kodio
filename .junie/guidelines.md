# Project Guidelines

This project is a multiplatform app for an AI-driven poker education platform.

## Project structure

- `flopiq-api`: shared models for network communication
- `flopiq-client`: client-side code
- `flopiq-server`: server-side code
- `flopiq-app:shared`: frontend code and business logic for interacting with the API
- `flopiq-app:app-android`: Android implementation
- `flopiq-app:app-ios`: iOS implementation
- `flopiq-app:app-desktop`: desktop implementation (JVM)
- `flopiq-app:app-web`: web implementation

Tests do not need to be run for routine changes.

You can ask Junie to regenerate these guidelines if the project structure changes.
